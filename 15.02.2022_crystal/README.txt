Код лежит в main.cpp, там в целом есть комментарии.
Есть две основные функции для вызова тестов - first_task_full_test и second_task_full_test;
Функции first_task_test и second_task_test являются промежуточными и трогать их не нужно.
========================================================
В first_task_full_test передается 5 параметров:
bool third_test - если хотим предельный случай одномерного кристалла, то ставим true;
int min_size, int max_size, int step - минимальный, максимальный размер кристалла и шаг изменения размера;
int num_of_simulations - количество симуляций при каждом размере доски;
========================================================
В second_task_full_test тоже передается 5 параметров:
bool third_test - если хотим предельный случай одномерного кристалла, то ставим true;
float min_percent, float max_percent, float step - минимальная, максимальная доля дислокаций и шаг изменения;
int num_of_simulations - количество симуляций при каждом размере доски;
========================================================
Измерения (сами циферки) лежат в first_task_data.txt - первое задание; second_task_data.txt - второе задание;
third_task_data_1.txt - первое задание, но с одномерным кристаллом;
third_task_data_2.txt - второе задание, но с одномерным кристаллом;
========================================================
Графики названы по аналогичному принципу, но к первому заданию (и его варинату с одномерным кристаллом) есть еще
вариант зависимости логарифмов измеренных величин.
